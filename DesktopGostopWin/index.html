맞고 (Desktop Gostop)

근무 중(?) 바탕화면 구석에서 몰래 즐기는 1:1~4인 화투 고스톱입니다. 브라우저에서도 플레이할 수 있지만, 투명도 조절·작업표시줄 숨김·전역 보스키 같은 스텔스 기능은 Windows 실행 파일에서만 동작합니다.

⬇ DesktopGostop.zip 다운로드 (약 14MB) ▶ 브라우저로 바로 플레이
설치 불필요 · 단일 exe · Windows 10 / 11

실행 방법

  • zip을 풀고 DesktopGostop.exe 실행 (Godot 설치 불필요).
  • SmartScreen 경고가 뜨면 추가 정보 → 실행 (서명 안 된 개인 빌드).
  • 실행하면 화면 우측 하단에 작은 위젯으로 뜹니다.

스텔스 기능 (습격 대비)

  • 투명도: 타이틀바 슬라이더로 창 전체를 실시간 반투명하게.
  • 작업표시줄 숨김: 숨 버튼 — 작업표시줄/Alt-Tab에서 사라짐.
  • 보스키(전역): 설에서 켜고 단축키 지정(기본 F9). 어디서든 그 키로 창 즉시 숨김/복원.
  • 위 항상 위 · 타이틀바 드래그로 이동.

게임

  • 인원 2~4인 선택, 빈 자리는 AI. 같은 월끼리 짝 맞춰 먹기(따닥!), 7점 나면 고/스톱.
  • 화면 크기에 맞춰 카드가 비례 확대됩니다.
