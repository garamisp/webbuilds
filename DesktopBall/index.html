DesktopBall

바탕화면 위에 뜬 투명 오버레이에서 공이 화면의 창들을 피해 등속으로 튕겨다니는 토이입니다. 브라우저에서는 실행되지 않는 Windows 실행 파일이라 내려받아 실행해야 합니다.

⬇ DesktopBall.zip 다운로드 (약 20MB)
.NET 설치 불필요 · Windows 10 / 11

실행 방법

  • zip을 풀고 DesktopBall.exe 와 win-scan.exe 를 같은 폴더에 둡니다.
  • DesktopBall.exe 실행.
  • SmartScreen 경고가 뜨면 추가 정보 → 실행 (서명 안 된 개인 빌드).

조작

  • 시스템 트레이(우측 하단)의 파란 원 우클릭: 공 개수 / 크기 / 그림 / 종료.
  • 종료 단축키: Ctrl + Alt + Q
  • 왼쪽 버튼을 누른 채 공에 가까이 가면 공이 밀려납니다. 클릭은 공을 통과합니다.
